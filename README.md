# Absolute Humidity Sensor

A custom Home Assistant integration that calculates absolute humidity from temperature and relative humidity sensors.

## Installation

1. Add this repository to HACS as a custom repository.
2. Install the "Absolute Humidity Sensor" integration.
3. Configure it via the UI by selecting your temperature and humidity sensors.

## Configuration

- **Temperature Sensor**: Select a sensor that provides temperature in °C.
- **Humidity Sensor**: Select a sensor that provides relative humidity in %.
